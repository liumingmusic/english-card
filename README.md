# 背单词 · 像百词斩一样刷词

一个**纯静态、零构建**的背单词软件，交互参考百词斩：**选单词本 → 开始背 → 一个一个过，每词做选择题自测 → 揭晓释义 → 标记「认识了 / 没记住」→ 进度条 → 一轮结束给统计**。也保留「浏览全部单词」的旧模式可翻看。

- 线上地址（部署后）：`https://<your-user>.github.io/english-card/`
- 无后端、无构建步骤、无任何 API key。
- **单词数据全部内置**：所有释义、音标、发音、例句、同反义词、用法都在发布时（`data/words.json`）一次性构建好，运行时**不联网、不调 API**，加载更快、可离线、永不空屏。

## ✨ 功能

- **选单词本**：首页卡片选择 全部 / 小学 / 初中 / 高中 / 四级 / 六级 / 考研 / 雅思 / 收藏 / 待复习，每张卡显示词数与「已掌握」进度。
- **开始背单词（百词斩式）**：
  - 进入背诵流，顶部进度条 +「第 X / N」。
  - 每个单词：大号单词、音标、🔊 发音、词性、英文释义、一条例句。
  - **选择题自测**：约一半概率「看单词选释义」，一半概率「看释义选单词」，四个选项，点完即时判对错（绿色=对，红色=你选错的）。
  - **揭晓详情**：释义、例句、同义词、反义词、用法说明。
  - **认识 / 没记住**：标记后自动进入下一个；也提供「下一个」跳过。标记写入 `localStorage`。
  - **结算页**：本轮「认识了 / 没记住 / 跳过」统计，可「再背一轮」「只背没记住的（待复习）」「返回选书」。
- **浏览全部单词（旧模式）**：双栏可滚动列表，点开看可滚动详情页，支持中英文搜索、收藏/掌握/复习标记。
- **发音**：优先用内置发音音频 URL；缺失则用浏览器内置 `speechSynthesis`（离线可用）。
- **收藏 / 掌握 / 待复习**：状态存 `localStorage`，「收藏」「待复习」也可作为独立单词本；掌握与复习互斥。

```
│   ├── seed.js            # 内置双语种子词表（四六级/考研/雅思）

├── .gitignore
└── README.md
```

## 🔌 数据源（免 key）

- **内置词表 `data/words.json`（随仓库发布）**：共 **437** 个单词，覆盖 7 个级别（小学 41 / 初中 103 / 高中 74 / 四级 99 / 六级 66 / 考研 31 / 雅思 23）。每条含单词、音标、发音音频、词性、中英释义、例句、同义词、反义词、用法说明。
- **Free Dictionary API**（免 key）：`https://api.dictionaryapi.dev/api/v2/entries/en/{word}`
  **仅在本地构建期**（`node scripts/build-words.js`）用于补全发音音频、英文释义、例句、同/反义词。**运行时零联网。**

## 🛠 本地开发

```bash
# 1) （可选）本地用 API 补全并重新生成 data/words.json（仅构建期联网）
npm run build

# 2) 启动一个静态服务器（任选）
npx serve .
# 或
python3 -m http.server 8080

# 3) 打开 http://localhost:8080
```

> 直接双击 `index.html` 也能用，但部分浏览器对 `file://` 的 fetch 有限制，建议用本地服务器。

## ☁️ 部署到 GitHub Pages

把仓库推到 `main` 分支即可，Pages 从 `main` 根目录提供静态文件。也可在仓库 Actions 里手动触发（若保留 workflow）。
